# Portfolio Implementation Plan

This project is a premium, high-impact UX/UI design portfolio inspired by modern dark-mode aesthetics. It features a bold layout, neon accents, and smooth scroll-based animations.

## Features
- **Modern Dark Theme**: Utilizes a charcoal background with neon lime accents and subtle grain texture.
- **Glassmorphism**: Applied to buttons, cards, and form inputs for a premium feel.
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices.
- **Scroll Animations**: Integrated with AOS (Animate On Scroll) for dynamic content entry.
- **Single Page Layout**: Smooth navigation between Home, Skills, Portfolio, Experience, and Contact sections.

## Technologies Used
- **Frontend**: HTML5, CSS3 (Custom Design System with Variables), Javascript (Vanilla).
- **Typography**: Montserrat (Headings), Inter (Body).
- **Icons**: Font Awesome 6.
- **Animations**: AOS Library.

## File Structure
- `index.html`: Main structure and content.
- `style.css`: Comprehensive design system and styling.
- `script.js`: Interaction logic and animation triggers.
- `assets/`: Folder containing hero portrait and project thumbnails.

## How to View
